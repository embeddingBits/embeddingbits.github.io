Software that I use and You Can Love

Software That I use

November 4, 2025   ·  1 min read

#Linux #Window Manager #Terminal #Text Editor #Utilities #Notes

Favourite #

Operating System: I use Arch.

Text Editor: I use Neovim and Zed.

Window Manager: I use River.

Terminal: I use Ghostty combined with fish shell.

Most of the softwares I use are made in Zig cause I love it so much :)

Utils #

File Manager: I use Yazi as my main and only file manager.

Web Browser: I use Firefox all the time.

PDF Viewer: I use Zathura as my pdf viewer and firefox sometimes.

Misc: For most of the other stuff like Word, Excel, etc…, I use LibreOffice.

College #

Editing Documents: I edit my documents in LateX and I use Neovim with a LateX compiler that compiles as I save the file.

Taking Notes: I manage my college and personal notes with Obsidian.
